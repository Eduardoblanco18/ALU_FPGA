# ALU_FPGA

Arithmetic Logic Unit (ALU) developed in VHDL for FPGA platforms.

This project was created for Digital Systems and Computer Architecture studies, focusing on modular hardware design, arithmetic and logical operations, and FPGA implementation.

---

## Features

- Arithmetic operations
  - Addition
  - Subtraction
  - Multiplication

- Logical operations
  - AND
  - OR
  - NOT

- Comparator support
- Seven-segment display output
- LED output visualization
- Modular VHDL architecture
- Testbench for simulation
